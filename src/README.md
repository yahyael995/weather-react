## ‚ú® Features

-   **City Search:** Find weather by city name.
-   **Geolocation:** Automatically get weather for your current location.
-   **15-Day Forecast:** Detailed forecast including temperature and weather icons.
-   **Dynamic Backgrounds:** The background image changes based on the weather (sunny, cloudy, rainy, etc.).
-   **Dark/Light Mode:** Switch between themes for comfortable viewing.
-   **Unit Conversion:** Toggle between Celsius and Fahrenheit.
-   **Debounced Search:** Smart search that waits for you to finish typing.
-   **Loading & Error States:** A professional UI that handles loading and network errors gracefully.
-   **Responsive Design:** Looks great on both desktop and mobile devices.
